# API WOMPO SHOP


## Installation

```bash
$ npm install
```

## Running docker compose in local
```bash
# development
docker-compose -f .\docker-compose-local.yml  up -d
```

## Running seed data
```bash
npm run seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
